# Real Alert 🚨 [English](README.md)
### מערכת OSINT לניתוח ואימות התראות צבע אדום בזמן אמת

**Real Alert** היא מערכת מבוססת פייתון שנועדה לנטר, לנתח ולאמת התראות על ירי רקטות בישראל. 

המערכת מתמחה בהבחנה בין "התראות מקדימות" (סימנים ראשונים) לבין "צבע אדום" (אזעקה בפועל), תוך שימוש במנוע לוגי-סטטיסטי לסינון רעשים וחיזוי סבירות פגיעה באזור **הרצליה, רמת השרון והסביבה**.

## 📍 התנסות בבוט
ניתן להתנסות במערכת בטלגרם: [@herzliya_alerts_bot](https://t.me/herzliya_alerts_bot)  
*(לתשומת לבכם: קבלת התראות Push בזמן אמת מחייבת אימות מיקום פיזי באזור הרצליה).*

## 🧠 תכונות עיקריות
- **ארכיטקטורת לקוח כפול**: שימוש ב-Telethon להאזנה לאותות גולמיים וב-Bot API להפצה למשתמשים.
- **אימות מבוסס מיקום (Geo-Fencing)**: מנגנון הצטרפות חכם המוודא רלוונטיות של המשתמשים ומונע עומס על השרת.
- **סינון סטטיסטי**: סינון אוטומטי של רעשים מאזורים מרוחקים (צפון/דרום) והתמקדות בווקטורים של גוש דן והשרון.
- **מנוע אימות (Validation)**: מערכת ביקורת עצמית המשווה בין החיזוי לבין האזעקות בפועל לשיפור מתמיד של האלגוריתם.
- **לוח בקרה למנהל**: ניהול משתמשים וצפייה בלוגים ישירות מתוך ממשק הטלגרם.

## 🛰 לוגיקת המיקום
כדי להבטיח דיוק מקסימלי, המערכת מגדירה "מלבן חוסם" (Bounding Box) כירורגי הכולל את:
* **נחלת עדה** (צפון)
* **רמת השרון** (דרום)
* **נווה עמל** (מזרח)
* **רחוב אילון** (מערב)



**פרטיות:** קואורדינטות ה-GPS מעובדות בזמן אמת לצורך אימות הגבול בלבד ו**אינן נשמרות** במאגרי המידע של המערכת.

- **ספריה מרכזית:** Telethon (MTProto API)
- **תשתית ענן:** Google Cloud Platform (GCP)
- **מערכת הפעלה:** Linux (Debian 12)
